# Compiling Erlang/OTP from source to snap

## Install
> snapcraft
> sudo snap erl
> sudo snap install erlang_22.1_amd64.snap --devmode

or

> snapcraft try 
> sudo snap try --devmode

## Shell
> snapcraft try --shell

## Run
> erlang
> elang.erl
> elang.erlc
> erlang.<TAB>

## Clean up
> sudo snap remove erlang
> sudo snapcraft clean
